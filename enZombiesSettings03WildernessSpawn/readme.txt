enZombie Settings Wilderness Spawn v1.1
https://github.com/ErrorNull0/enZombiesSettings

Modify the number of zombies that radnomly spawn outside in the wilderness in each biome. Also, modify the daily re-spawn rate of those zombies.

The mod is intended for use with the main enZombies mod.
