enZombie Settings Attack Range v1.1
https://github.com/ErrorNull0/enZombiesSettings

Adjust the hand-to-hand / melee attack distance of each enZombie. This does not alter any vanilla zombies.

The mod is intended for use with the main enZombies mod.

enzombies mod page:
